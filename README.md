# 🤖 AI-Powered Resume Screening Platform

An intelligent ATS (Applicant Tracking System) built using Python, NLP, Machine Learning, and Streamlit for automated resume screening, candidate ranking, and recruitment analytics.

## 🚀 Features
- PDF & DOCX resume parsing
- TF-IDF + cosine similarity matching
- Skill-based ATS scoring
- Candidate ranking & shortlisting
- AI-powered recruiter dashboard
- Recruitment analytics & insights
- CSV report generation

## 🧠 Tech Stack
- Python
- Pandas & NumPy
- Scikit-learn
- pdfplumber
- python-docx
- Streamlit
- Plotly
